CSS и HTML - это два разных языка для разных целей. При написании html-кода для CSS, выбирая теги, ориентируйтесь на роль,
которую играет фрагмент текста на веб странице, а не на внешний вид,
который текст приобретает благодаря этому тегу.
Применяя CSS для дизайна веб-страницы, вы используете HTML только по его прямому назначению,
то есть именно для разметки веб-страницы на логические фрагменты,
не заботясь о форматировании и внешнем виде страницы.
