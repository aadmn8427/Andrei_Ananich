CSS свойство border-image используется для указания значений таких свойств, как: border-image-source, border-image-width, border-image-slice, border-image-outset и border-image-repeat в одном объявлении. Данные свойства можно использовать, например для создания красивых кнопок. Если какое-либо значение было не установлено, то оно будет установлено в значение по умолчанию для соответствующего свойства.
